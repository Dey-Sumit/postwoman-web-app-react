# PostWoman 👩‍🦰

### PostWoman is a postman clone (made with react and love 💙) that runs on web. 


![](https://raw.githubusercontent.com/Dey-Sumit/postwoman-web-app-react/main/public/github/home.PNG)

## Installation

```bash
npm run start
```
